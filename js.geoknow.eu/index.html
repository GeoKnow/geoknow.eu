Welcome to VeriLlinks. Turn the world into a better place by playing games ;-)

How?
VeriLinks is a light-weight game platform with the purpose of validating links of the Web of Data. A task that even allmighty computer algorithms can't solve completely. That's why we need your help! Read More
List of Games:

Pea Invasion

Do you have multitasking skills? Try to defend from evil peas, while answering a quiz.

Space Ships

Play this turn based quiz game. Defeat your opponent by correctly answering as many questions as possible.

Features

What can VeriLinks do for you?

  • System for managing, retrieving, displaying and verifying Semantic Web Links
  • Central Server with communication API
  • Core functionality for game administration
  • User and game statistic management
  • Rewarding and balancing system
  • more features

Use VeriLinks in your game

Integrate our Javascript Widget into your own game to unlock all the features of VeriLinks.

Take a look at our tutorial on how to integrate and utilize the Widget.
